# TEST CASES

| ID | Тест-кейс | Входные данные | Ожидаемый результат | Статус |
|---|---|---|---|---|
| TC-01 | Поиск с корректными параметрами | `GET /api/products?type=ВЦ&minAirflow=1000&maxAirflow=5000&sort=price_asc&limit=48&offset=0` | Возвращается `200 OK`, в `items` есть модели, соответствующие фильтрам | Not Run |
| TC-02 | Поиск с параметрами, которых нет в базе | `GET /api/products?type=НЕСУЩЕСТВУЮЩИЙ_ТИП&minAirflow=999999&maxAirflow=1000000` | Возвращается `200 OK`, `items` пустой; UI показывает пустой список или блок аналогов | Not Run |
| TC-03 | Поиск с пустыми полями | `GET /api/products?minAirflow=&maxAirflow=&limit=48&offset=0` | Возвращается `422 Unprocessable Entity` с описанием ошибки валидации параметров | Not Run |
| TC-04 | Поиск с отрицательным значением м³/ч | `GET /api/products?minAirflow=-100&limit=48&offset=0` | Возвращается ошибка валидации (`422`) для отрицательного значения | Not Run |
| TC-05 | Поиск только по типу | `GET /api/products?type=ВЦ&limit=48&offset=0` | Возвращается `200 OK`, список содержит все модели выбранного типа с учетом пагинации | Not Run |
| TC-06 | Запрос PDF для одной модели | На странице сравнения выбрать 1 модель и нажать экспорт PDF | PDF-файл скачивается, содержит данные выбранной модели | Not Run |
| TC-07 | Запрос PDF для трех моделей | На странице сравнения выбрать 3 модели и нажать экспорт PDF | PDF-файл скачивается, содержит все три модели и их параметры | Not Run |
| TC-08 | Одновременный поиск тремя пользователями | 3 клиента одновременно отправляют `GET /api/products` с разными фильтрами | Сервер отвечает всем запросам, ошибок `5xx` нет, приложение не падает | Not Run |
| TC-09 | Поиск при отключенном интернете | Клиент в локальной сети открывает UI и выполняет поиск при отключенном внешнем интернете | Поиск работает штатно (локальный API доступен, ответы приходят) | Not Run |
| TC-10 | Поиск на 1000 моделей (производительность) | Запрос к выборке большого объема (например, `limit=1000`) и замер времени ответа | Время ответа API не превышает 2 секунды | Not Run |
