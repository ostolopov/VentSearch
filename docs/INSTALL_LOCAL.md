# 🚀 Инструкция по установке и запуску VentSearch

**Для ООО «ЗАВОД ВЕНТМАШ»**  
*Версия для конечного пользователя (Windows 10/11)*

---

## 📋 Что нужно перед началом

| Компонент        | Требование / ссылка                                                    |
| ---------------- | ---------------------------------------------------------------------- |
| Операционная система | Windows 10 (64-разрядная) или Windows 11                            |
| Docker Desktop   | [Скачать с официального сайта](https://www.docker.com/products/docker-desktop/) |
| Память и место   | Минимум 4 ГБ ОЗУ, 500 МБ свободного места на диске                    |
| Права            | Учётная запись с правами администратора (для установки)               |

> ⚠️ **Важно:** Docker — это изолированная среда, которая не изменяет системные файлы Windows и не конфликтует с другими программами. После запуска система работает автономно на вашем компьютере.

---

## 1️⃣ Установка Docker (делается **один раз**)

### Шаг 1. Скачайте установщик
Перейдите по ссылке и нажмите **Download Docker Desktop**:  
🔗 [https://www.docker.com/products/docker-desktop/](https://www.docker.com/products/docker-desktop/)

### Шаг 2. Установите
- Запустите скачанный файл и следуйте мастеру установки (кнопки «Далее» / «Install»).
- По окончании **обязательно перезагрузите компьютер**.

### Шаг 3. Запустите Docker
- Найдите в меню «Пуск» **Docker Desktop** и откройте его.
- Дождитесь, пока значок кита в системном трее (рядом с часами) станет **зелёным** и появится надпись «Engine running».
- Это может занять 1–2 минуты. Не закрывайте окно Docker – оно должно работать в фоне.

---

## 2️⃣ Получение проекта VentSearch

1. Откройте браузер и перейдите в репозиторий:  
   🔗 [github.com/ostolopov/VentSearch](https://github.com/ostolopov/VentSearch)
2. Нажмите большую зелёную кнопку **Code**, затем выберите **Download ZIP**.
3. Сохраните архив и распакуйте его в любую удобную папку (например, `C:\VentSearch`).

---

## 3️⃣ Запуск системы

### Шаг 1. Подготовка
- Убедитесь, что **Docker Desktop запущен** (зелёный значок кита).
- Откройте папку с распакованным проектом.

### Шаг 2. Запустите `run.bat`
- Найдите в папке файл **`run.bat`** и запустите его двойным щелчком мыши.
- Появится чёрное окно консоли – это нормально. **Не закрывайте его**, пока работаете с программой.

> ℹ️ **При первом запуске** Docker автоматически скачает необходимые компоненты (образы PostgreSQL, Python). Это займёт до 5 минут и зависит от скорости интернета. В следующий раз запуск будет почти мгновенным.

### Шаг 3. Дождитесь готовности
В консоли появятся системные логи. Когда вы увидите строку наподобие  
`Uvicorn running on http://0.0.0.0:8000` – сервер готов к работе.

---

## 4️⃣ Открытие интерфейса VentSearch

### 🌐 Основной способ (на этом же компьютере)
Откройте браузер и введите адрес:  
🔗 **`http://127.0.0.1:8000`** (или просто кликните по ссылке).

Вы попадёте в веб-интерфейс поиска вентиляторов.

---

### 📡 Доступ с любого устройства в вашей локальной сети (Wi‑Fi / LAN)

Теперь **самое полезное**: VentSearch можно открыть не только на компьютере, где он запущен, но и с других устройств, подключённых к тому же роутеру (смартфон, планшет, ноутбук, ещё один ПК).  
Это особенно удобно – например, инженер в цехе может открыть подбор с планшета.
