# 🚀 Инструкция по установке и запуску VentSearch

Система подбора промышленных вентиляторов для ООО «ЗАВОД ВЕНТМАШ».

---

## 📋 Предварительные требования
Для работы программы вам понадобится:
1.  **Операционная система:** Windows 10 или Windows 11.
2.  **Docker Desktop:** Инструмент для запуска программы в изолированной среде (гарантирует работу без ошибок на любом ПК).

---

## 1. Установка окружения (делается один раз)

### Шаг 1: Скачивание Docker
Перейдите на официальный сайт и скачайте установщик:
👉 [Скачать Docker Desktop](https://www.docker.com/products/docker-desktop/)

### Шаг 2: Установка и настройка
1. Запустите скачанный файл и следуйте инструкциям (нажимайте «Next» / «Ok»).
2. **Обязательно перезагрузите компьютер** после завершения установки.
3. Запустите Docker Desktop из меню «Пуск». Подождите 1-2 минуты, пока значок кита в нижнем углу не станет зеленым.

---

## 2. Скачивание и запуск проекта

### Шаг 3: Получение файлов проекта
1. Перейдите в репозиторий: [github.com/ostolopov/VentSearch](https://github.com/ostolopov/VentSearch)
2. Нажмите зеленую кнопку **Code** -> **Download ZIP**.
3. Распакуйте архив в любую удобную папку на вашем компьютере.

### Шаг 4: Запуск системы
1. Убедитесь, что **Docker Desktop открыт** (это важно!).
2. Зайдите в папку с проектом.
3. Найдите файл `run.bat` и запустите его двойным кликом.
   * *Примечание: При первом запуске система скачает необходимые компоненты (PostgreSQL, Python-образ). Это может занять до 5 минут.*
4. Не закрывайте появившееся черное окно консоли, пока работаете в программе.

---

## 3. Работа с программой

После того как в консоли появятся системные логи (сообщения о запуске), откройте браузер и введите в адресную строку:

🔗 **[http://127.0.0.1:8000/](http://127.0.0.1:8000/)**

Перед вами откроется веб-интерфейс VentSearch с каталогом и функцией поиска.

---

## 🛠 Техническая информация (для IT-специалистов)

Если вы предпочитаете ручное управление через терминал:

1. **Клонирование:** `git clone https://github.com/ostolopov/VentSearch.git`
2. **Развертывание:** `docker-compose up --build`
3. **Порты:**
   * Backend/Frontend: `8000`
   * Database (PostgreSQL): `5432`
4. **Документация API (Swagger):** `http://127.0.0.1:8000/docs`

---

## ❓ Решение проблем
* **Проблема:** Сайт не открывается.
  * *Решение:* Проверьте в Docker Desktop (вкладка Containers), что контейнер `ventsearch_app` имеет статус **Running**.
* **Проблема:** Ошибка "Docker is not running".
  * *Решение:* Запустите приложение Docker Desktop и дождитесь его полной загрузки.
