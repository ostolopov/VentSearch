# Swagger UI (/docs)

1. Запустите проект: `./run.sh`
2. Откройте в браузере: `http://127.0.0.1:<PORT>/docs` (порт из `.env`, по умолчанию 5000).
3. Альтернатива: ReDoc — `http://127.0.0.1:<PORT>/redoc`.

Для задачи в трекере: сделайте скриншот страницы `/docs` с раскрытыми описаниями эндпоинтов и прикрепите файл к задаче.
